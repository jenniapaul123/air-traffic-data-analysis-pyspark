# Air Traffic Data Analysis using PySpark

##  Overview
This project demonstrates core Spark DataFrame operations using PySpark on an air traffic dataset.  
It focuses on data loading, schema understanding, transformation, filtering, aggregation, and analysis.

The project is implemented in a local environment and simulates distributed data processing concepts similar to workflows performed using AWS EMR.

## 🎯 Objectives
- Load structured flight data using PySpark
- Define and inspect schema
- Perform data transformations using Spark DataFrames
- Apply filtering, sorting, and aggregation operations
- Analyze distinct origin and destination countries
- Demonstrate sampling and repartitioning
  

## Technologies Used
- Python
- PySpark
- Jupyter Notebook
- Spark DataFrames
